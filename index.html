git和github的使用

GitLens

GitHistory

新建终端：

git init

git add ./*.py

git commit -m “xxxx”

git remote add origin https://github.com/cumter-yan/Python_OOP.git

git push -u origin master

工作区 缓冲区 归档区 push
7.py 7.py 7.py 远程仓库

git status—–查看本地仓库的状态

git log—–查看提交以及修改信息

git reset –mixed 47b080770f052164987bbb3f18940754c10266b0———归档区和缓冲区回滚

git reflog——查看版本信息

git reset –hard 2b4f136————–再次还原

git reset –soft 47b080770f052164987bbb3f18940754c10266b0—–归档区回滚

git revert 2b4f136245f31843fc14aa9607ad7e593487b346

“修改了6.py”内容被回滚

查看当前分支

新建一个分支b1并切换到此分支

将b1分支合并到master分支

git pull将本地版本更新和远端仓库版本一致
